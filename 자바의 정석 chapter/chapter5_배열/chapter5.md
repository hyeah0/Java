### 자바의 정석 chapter 5 배열

1. 배열이란?

같은 타입의 여러변수를 하나의 묶음으로 다루는것

2.  배열의 선언과 생성

    1. 배열 선언

    - 타입[] 변수이름;

             예시)
             int[] score;
             String[] naem;

    - 타입 변수이름[];

            예시)
            int score[];
            String name[];

    2. 배열 생성

    - 타입[] 변수이름;

      변수이름 = new 타입[길이];

            예시)
            int[] score;
            score = new int[5];

    - 타입[] 변수이름 = new 타입[길이]

            예시)
            int[] score = new int[5]
